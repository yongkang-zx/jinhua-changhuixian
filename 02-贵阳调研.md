## 一、长期护理保险政策

### 1.1 国家背景（核心 A 级信源）

- **中办国办 2025-03-25《关于加快建立长期护理保险制度的意见》** —— 标志长护险从局部试点转向全国推行
  - 主要目标：3 年左右时间统筹城乡的制度安排基本确立；**到 2028 年底，长期护理保险制度将在全国范围基本实现全覆盖**
  - 筹资：长期护理保险费率 **统一控制在 0.3% 左右**
  - 待遇：基金支付比例按未就业城乡居民 50% 左右 / 单位职工 70% 左右
  - 原文：[甘肃纪检监察网（中办国办原文转载）](http://www.gsjw.gov.cn/content/2026-03/71139.html)
  - 解读：[四川省人民政府 2026-07-19 转载新华社](https://www.sc.gov.cn/10462/13241/2026/3/26/479495ef60f94c2a96a5255af8b06471.shtml)

### 1.2 贵州省实施意见（核心 A 级信源）

- **省政府办公厅《关于加快建立长期护理保险制度的实施意见》**
  - **2028 年 12 月底前，全省基本建立长期护理保险制度**（覆盖全民、统筹城乡、互助共济、保障基本、公平适度、规范统一）
  - 来源：[贵州省政府·贵州长护险 2028 年底全覆盖](https://so.html5.qq.com/page/real/search_news?docid=70000021_1756a2cb58c21152)
  - 解读：[贵州省政府·2028 年底基本实现全省全覆盖](https://so.html5.qq.com/page/real/search_news?docid=70000021_2886a292edf42652)
  - 政策原文：[贵州省医保局政策法规库](https://m12333.cn/platform/raz.html)（[2026-06-09](https://m12333.cn/platform/raz.html)《关于加快建立长期护理保险制度的实施意见》）

### 1.3 贵州省实施意见关键政策要点

| 维度 | 内容 |
|---|---|
| **覆盖范围** | 用人单位及其职工、退休人员、灵活就业人员和未就业城乡居民等均可参保；起步阶段优先覆盖单位职工、退休人员和灵活就业人员，再逐步将未就业城乡居民纳入 |
| **统筹层级** | **市（州）级统筹**，基金统收统支；条件成熟后推动省级统筹 |
| **筹资费率** | 全省统一 **0.3% 左右** |
| **单位职工** | 个人 0.15% + 单位 0.15%，各以本人工资/职工工资总额为基数 |
| **退休人员** | 0.15%，以本人基本养老金为基数 |
| **未就业居民** | 0.3%（1:1 个人与政府补助）；新启动地区 0.15% 起步，5 年过渡到 0.3% |
| **灵活就业** | 可选按单位职工或未就业居民政策 |
| **个账使用** | 职工医保个人账户可用于本人及近亲属（配偶/父母/子女/兄弟姐妹/祖父母/外祖父母/孙子女/外孙子女）参加长护险的个人缴费 |
| **待遇支付** | 不设起付线；职工/退休 70% 左右；未就业居民 50% 左右；基金年度最高支付限额 ≤ 上年度统筹地区城乡居民人均可支配收入的 50% |
| **重点保障** | 起步阶段重点保障 **重度失能人员**，未来逐步扩大 |
| **评估标准** | 全国统一的失能等级评估标准，评估结果 **全省互认**，适时推动全国互认 |
| **经办平台** | 依托全国统一医保信息平台，实现医保/税务/民政/卫健/人社/残联等部门信息共享，支持"一网通办" |

### 1.4 贵州省试点进程

| 时间 | 城市 | 状态 |
|---|---|---|
| 2020-09 | **黔西南州** | 列为全国第二批试点（第二个少数民族自治地区）|
| 2021 | 黔西南州 | 在全州城镇职工基本医疗保险参保人员范围内开展试点 |
| 2022-10 | 黔西南州 | 兴义市、兴仁市城乡居民基本医疗保险参保人员纳入 |
| 2024-01-24 | 黔西南州 | 2024 年长护险工作部署会：1 家经办机构 + 1 家辅具机构 + 12 家上门护理 + 5 家养老服务 |
| 截至 2024-01 | 黔西南州 | 累计享受待遇 1856 人，60+ 失能 1484 人（80%），居家 1262 人（68.82%），人均年支付 1.11 万元 |
| 2026 | **遵义市、铜仁市** | 已启动实施 |
| [待补证] | **贵阳市** | 是否纳入下一批启动时点 |

## 二、养老政策

### 2.1 贵州省经济困难失能老年人等群体集中照护

- **2025-03 印发《关于做好经济困难失能老年人等群体集中照护服务工作的实施意见》**
  - **服务对象**：低保失能老年人 + 高龄老年人（80+）
  - **保障标准**：不超当地特困人员基本生活费 + 全护理照料费总额
  - 来源：[贵州省民政厅](https://m12333.cn/platform/raz.html)

### 2.2 2026 年贵州省失能老年人"健康敲门行动"

- **服务对象**：**12 万名 65+ 失能老年人**
- **服务内容**：免费上门服务
- 来源：[贵州省民政厅 2026 年方案](https://m12333.cn/platform/raz.html)

### 2.3 贵阳市养老服务体系（核心数据）

- **60+ 老年人口**：**82.7 万人**
- **老龄化率**：**13.3%**（2023-10 截至）
- **养老服务机构/设施**：**960 个**
- **床位**：**2.4 万余张**
- **备案机构护理型床位占比**：**56%**
- **街道养老服务中心**：**25 个**
- **社区养老服务站**：**264 个**
- **工作思路**："**12345**"（1 个核心 + 2 个抓手 + 3 个支撑 + 4 个保障 + 5 个场景）
- **服务网络**：**15 分钟养老服务圈**

### 2.4 贵州省养老服务发展目标

- "十四五"末：每个县（市、区）至少有 1 所老年大学
- 长期照护师培训培养机制建立
- 职业院校培养专业护理人才
- 信息化、智能化技术与长护险服务融合

### 2.5 政策衔接

- 贵州省 2025-01-07《中共中央 国务院关于深化养老服务改革发展的意见》：加快建立长护险制度，做好长护险与经济困难失能老年人护理补贴的衔接；推广包含长护险责任的商业健康保险产品
  - 来源：[搜狐财经 2025-01-07](https://www.sohu.com/a/846392035_115865)
